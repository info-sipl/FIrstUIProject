# First UI Project

A starter template for building modern web applications with HTML, CSS, and JavaScript.

## Project Structure

```
FIrstUIProject/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Stylesheet
├── js/
│   └── script.js       # JavaScript file
├── images/             # Folder for images
└── README.md           # Project documentation
```

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/info-sipl/FIrstUIProject.git
   cd FIrstUIProject
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your web browser
   - Or use a live server extension in your code editor

## Features

- ✨ Responsive design that works on all devices
- 🎨 Modern CSS styling with gradients and animations
- ⚡ Interactive JavaScript functionality
- 📁 Organized project structure

## Customization

- **CSS:** Edit `css/style.css` to change colors, fonts, and layouts
- **HTML:** Modify `index.html` to add your own content
- **JavaScript:** Update `js/script.js` to add interactive features
- **Images:** Add your images to the `images/` folder

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)
